# Flowlua

To get started.

```
local dir="$HOME/flowtools/cosmopolitan" 
mkdir -p   $dir
pushd      $dir
curl -L -O https://cosmo.zip/pub/cosmocc/cosmocc.zip
unzip cosmocc.zip 
rm    cosmocc.zip
popd 
```


```
git clone https://github.com/eliotmuirgrid/flowlua
```

```
cd flowlua/flowlua
```

```
make
```
